# 📖 Overview

This repository contains a well-organized collection of **Object-Oriented Programming (OOP)** programs written in **C++**. It is designed for students, beginners, and programming enthusiasts who want to understand the principles of OOP through practical examples.

The repository covers everything from basic classes and objects to advanced topics such as inheritance, polymorphism, abstraction, templates, and file handling.

# 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/oop-concepts-cpp.git
```

### Compile

```bash
g++ filename.cpp -o program
```

### Run

**Windows**

```bash
program.exe
```

**Linux / macOS**

```bash
./program
```
